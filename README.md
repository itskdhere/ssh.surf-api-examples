# SSH.SURF API Examples

Some examples of how to use the SSH.SURF API and SSH.SURF API Wrapper ([ssh.surf](https://github.com/itskdhere/ssh.surf)).

### Requirements:

- Node.js v18.16.x or higher
- NPM v9.x.x or higher

### Usage:

- Clone this repository:

```bash
git clone https://github.com/itskdhere/ssh.surf-api-examples
```

```bash
cd ssh.surf-api-examples
```

- Install dependencies:

```bash
npm install
```

- Create a .env file and add your SSH.SURF API key:

```bash
cp .env.example .env
```

```bash
nano .env
```

- Run the examples:

```bash
npm run hello
```

```bash
npm run name
```

```bash
npm run start
```

```bash
npm run stop
```

```bash
npm run restart
```

```bash
npm run info
```

```bash
npm run stats
```

```bash
npm run time
```

```bash
npm run newRootPass
```

```bash
npm run newKey
```

```bash
npm run keyTime
```

```bash

```

```bash

```

```bash

```

